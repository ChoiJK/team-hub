# Rules: ELT (EnglishLearningToolkit)

## 스택
- Next.js (App Router) + TypeScript + Tailwind CSS
- Supabase (DB)
- Pretendard 폰트

## 디자인
- Garden Rest 테마 (차분한 녹색/베이지)
- Primary: #4A7C59
- Background: #F5F0E8
- Discord 스타일 3-column 레이아웃

## 원칙
- 레벨 ≠ 콘텐츠 수준 — 기초 학습도 성인 문맥
- 죄책감 없는 학습 — 부정적 표현 금지
- Mobile-first responsive
